# Cms from scratch

Create a cms in Poo from scratch

# Installation

Installation :

- Webserver Apache2
- Php >8.2
- Mysql / MariaDb
- composer
- twig (via composer)
- php-dotenv (via composer)
- php-cs fixer, set to psr-12 (optionnal)
- import database located in /docs/database in phpMyAdmin
- git clone https://github.com/FractalFramework/oc5.git
